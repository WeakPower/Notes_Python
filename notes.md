### 面向对象

* 基本特征：抽象（abstract）、继承（inherit）、多态（Polymorphism）
* 实例方法、类方法（@claasmethod）、静态方法（@staticmethod）
* 构造设计{JAVA、C++等} = 创建（__new__） + 初始化（__init__） {python} 

### 异常处理

* except Exception as ret : 
		print (ret)
* else / finally
* ""(空字符)、None、[]、{} 、 0都是为假，其它有值（如非0）均为真

### 模块使用

* 主程序不调用子模块测试部分：
	if __name__ == '__main__':
		main()
